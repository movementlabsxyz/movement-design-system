# borderWidths

A scale for border width values.

## Import

```tsx
import { borderWidths } from "@movementlabsxyz/movement-design-system";
```

## Available Values

`0` through `8` (0px, 1px, 2px, 3px, 4px, 5px, 6px, 7px, 8px)

## Basic Usage

```tsx
import { borderWidths } from "@movementlabsxyz/movement-design-system";

console.log(borderWidths[1].value); // "1px"
console.log(borderWidths[4].value); // "4px"
```

## Outlined Box

```tsx
import { borderWidths } from "@movementlabsxyz/movement-design-system";

export function OutlinedBox() {
  return (
    <div style={{ 
      borderWidth: borderWidths[2].value, 
      borderStyle: "solid",
      borderColor: "#ccc",
      padding: "16px"
    }}>
      Box with 2px border
    </div>
  );
}
```

## Thick Border

```tsx
import { borderWidths } from "@movementlabsxyz/movement-design-system";

export function ThickBorder() {
  return (
    <div style={{
      borderWidth: borderWidths[4].value,
      borderStyle: "solid",
      borderColor: "#000",
    }}>
      Element with thick border
    </div>
  );
}
```

## Mixed Borders

```tsx
import { borderWidths } from "@movementlabsxyz/movement-design-system";

export function MixedBorders() {
  return (
    <div style={{
      borderTopWidth: borderWidths[2].value,
      borderBottomWidth: borderWidths[1].value,
      borderLeftWidth: borderWidths[0].value,
      borderRightWidth: borderWidths[0].value,
      borderStyle: "solid",
      borderColor: "#666",
    }}>
      Element with top and bottom borders
    </div>
  );
}
```

## Conditional Borders

```tsx
import { borderWidths } from "@movementlabsxyz/movement-design-system";

export function ConditionalBorder({ isActive }: { isActive: boolean }) {
  return (
    <div style={{
      borderWidth: isActive ? borderWidths[2].value : borderWidths[1].value,
      borderStyle: "solid",
      borderColor: isActive ? "#0066ff" : "#ccc",
    }}>
      Active state border
    </div>
  );
}
```

## Common Uses

| Value | Size | Common Use |
|-------|------|------------|
| `borderWidths[0]` | 0px | No border |
| `borderWidths[1]` | 1px | Subtle borders |
| `borderWidths[2]` | 2px | Standard borders |
| `borderWidths[3]` | 3px | Emphasized borders |
| `borderWidths[4]` | 4px | Bold borders |
| `borderWidths[8]` | 8px | Very bold borders |
