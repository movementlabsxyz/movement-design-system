# shadows

A scale for box shadow values, from subtle to dramatic.

## Import

```tsx
import { shadows } from "@movementlabsxyz/movement-design-system";
```

## Available Values

- `xs`: Smallest shadow (hairline)
- `sm`: Small shadow
- `base`: Default shadow
- `md`: Medium shadow
- `lg`: Large shadow
- `xl`: Extra large shadow
- `2xl`: Dramatic shadow
- `inner`: Inner shadow
- `none`: No shadow

## Basic Usage

```tsx
import { shadows } from "@movementlabsxyz/movement-design-system";

console.log(shadows.sm.value);  // Subtle shadow
console.log(shadows.lg.value);  // Large shadow
```

## Card with Shadow

```tsx
import { shadows } from "@movementlabsxyz/movement-design-system";

export function Card({ elevated }: { elevated?: boolean }) {
  return (
    <div style={{ 
      boxShadow: elevated ? shadows.lg.value : shadows.base.value,
      padding: "16px",
      borderRadius: "8px"
    }}>
      Card with shadow
    </div>
  );
}
```

## Inner Shadow

```tsx
import { shadows } from "@movementlabsxyz/movement-design-system";

export function InsetPanel() {

    </div>

```

## Hover Effect

```tsx
import { shadows } from "@movementlabsxyz/movement-design-system";
import { useState } from "react";

export function HoverCard() {
  const [isHovered, setIsHovered] = useState(false);
  
  return (
    <div 
      style={{ 
        boxShadow: isHovered ? shadows.xl.value : shadows.md.value,
        padding: "16px",
        transition: "box-shadow 0.2s"
      }}
      onMouseEnter={() => setIsHovered(true)}
      onMouseLeave={() => setIsHovered(false)}
    >
      Hover for shadow effect
    </div>
  );
}
```

## Modal with Dramatic Shadow

```tsx
import { shadows } from "@movementlabsxyz/movement-design-system";

export function Modal() {
  return (
    <div style={{
      boxShadow: shadows["2xl"].value,
      padding: "32px",
      borderRadius: "12px",
      backgroundColor: "white"
    }}>
      Modal with dramatic shadow
    </div>
  );
}
```

## Shadow Scale

| Value | Effect | Common Use |
|-------|--------|------------|
| `shadows.xs` | Minimal | Subtle separation |
| `shadows.sm` | Small | List items, small cards |
| `shadows.base` | Default | Cards, panels |
| `shadows.md` | Medium | Elevated cards |
| `shadows.lg` | Large | Floating elements |
| `shadows.xl` | Extra large | Popovers, dropdowns |
| `shadows.2xl` | Dramatic | Modals, overlays |
| `shadows.inner` | Inset | Input wells, inset panels |
| `shadows.none` | None | Remove shadows |
