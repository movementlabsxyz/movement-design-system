# sizes

A scale for sizing values (width, height, max-width, etc.). Includes all spacing values plus semantic size tokens.

## Import

```tsx
import { sizes } from "@movementlabsxyz/movement-design-system";
```

## Available Values

- All values from `spacing` (0-100, px)
- `prose`: 65ch (optimal reading width)
- `full`: 100%
- `min`: min-content
- `max`: max-content
- `fit`: fit-content

## Basic Usage

```tsx
import { sizes } from "@movementlabsxyz/movement-design-system";

console.log(sizes.prose.value); // "65ch"
console.log(sizes.full.value);  // "100%"
console.log(sizes[64].value);   // "256px"
```

## Prose Width

```tsx
import { sizes } from "@movementlabsxyz/movement-design-system";

export function ProseContainer({ children }: { children: React.ReactNode }) {
  return (
    <div style={{ maxWidth: sizes.prose.value, margin: "0 auto" }}>
      {children}
    </div>
  );
}
```

## Full Width

```tsx
import { sizes } from "@movementlabsxyz/movement-design-system";

export function FullWidthSection() {
  return (
    <div style={{ width: sizes.full.value }}>
      Full width section
    </div>
  );
}
```

## Fixed Sizes

```tsx
import { sizes } from "@movementlabsxyz/movement-design-system";

export function Avatar() {
  return (
    <div style={{
      width: sizes[10].value,   // 40px
      height: sizes[10].value,  // 40px
      borderRadius: "50%"
    }}>
      Avatar
    </div>
  );
}
```

## Content-Based Sizing

```tsx
import { sizes } from "@movementlabsxyz/movement-design-system";

export function FlexibleButton() {
  return (
    <button style={{ width: sizes.fit.value, padding: "8px 16px" }}>
      Fits content
    </button>
  );
}
```

## Semantic Sizes

| Value | Meaning | Common Use |
|-------|---------|------------|
| `sizes.prose` | 65ch | Article/blog content |
| `sizes.full` | 100% | Full width containers |
| `sizes.min` | min-content | Minimal width |
| `sizes.max` | max-content | Content-driven width |
| `sizes.fit` | fit-content | Fit to content |
