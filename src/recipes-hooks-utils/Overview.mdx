## ðŸ’¡ Pro Tips

1. **Use `cn` for all class merging** - It intelligently handles Tailwind class conflicts
2. **Recipes are composable** - Combine them with your own classes using `cn`
3. **Theme scales work with inline styles** - Perfect when you need JS values
4. **Hooks handle SSR** - `useIsMobile` returns `undefined` initially to prevent hydration issues
5. **Types are exported** - Use `VariantProps<typeof buttonVariants>` for type safety
