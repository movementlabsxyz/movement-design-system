# Colors

The available color tokens that can be applied to color-related properties like `color`, `bg`, and `borderColor` in PandaCSS.

{/* prettier-ignore-start */}

<Source code={`
import { css } from "styled-system/css";

const MyComponent = () => (
  <p
    className={css({
      color: "protocol-pink.500",
      bg: "byzantine-blue.400", 
      borderColor: "guild-green.400",
    })}
  >
    Hello World
  </p>
);
`} />
